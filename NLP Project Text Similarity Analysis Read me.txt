NLP Project: Text Similarity Analysis

🔹 Overview
This project implements a comprehensive Natural Language Processing (NLP) pipeline to measure text similarity at three levels: word, sentence, and document. It combines classical NLP techniques with modern deep learning models to provide accurate semantic similarity analysis.

🔹 Objectives
Compute word-level similarity using Edit Distance, WordNet, and Word2Vec
Measure sentence-level similarity using Jaccard, TF-IDF, and SBERT
Analyze document-level similarity using TF-IDF and Doc2Vec
Build a unified pipeline for automatic similarity detection

🔹 Methods Implemented

✅ Word Similarity
Edit Distance (Lexical similarity)
WordNet Path Similarity (Semantic)
Word2Vec / GloVe (Contextual similarity)

✅ Sentence Similarity
Jaccard Similarity
TF-IDF + Cosine Similarity
SBERT Embeddings (Deep semantic similarity)

✅ Document Similarity
TF-IDF Vectorization
Doc2Vec Embeddings
SBERT (for semantic retrieval)

🔹 Key Features
Multi-level similarity analysis (word, sentence, document)
Automatic method selection using pipeline
Visualization using heatmaps
Semantic search using SBERT

🔹 Results & Insights
Edit Distance is useful for spelling but fails for meaning
WordNet captures lexical relationships but has limitations
Word2Vec & SBERT provide strong semantic understanding
SBERT performs best for sentence and document similarity tasks

🔹 How to Run
1. Install Dependencies
pip install nltk spacy scikit-learn sentence-transformers gensim matplotlib seaborn
2. Download Models
python -m spacy download en_core_web_md
python -m nltk.downloader wordnet punkt omw-1.4
3. Run the Notebook
jupyter notebook NLP.ipynb
